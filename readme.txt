== Description ==

A Wordpress plugin, made to customize the admin & login areas of a website.

Features:

*   Add custom dashboard classes
*   Add custom CSS style to dashboard
*   Modify the footer text for the dashboard
*   Remove dashboard widgets
*   Custom login page
*   Add your own logo to the admin bar

== Installation ==

1. Download
2. Unzip
3. Upload `good-dashboard` to the `/wp-content/plugins/` directory
4. Activate the plugin through the 'Plugins' menu in WordPress
